Commands:

- `tsargp -h` print the help message
- `tsargp -v` print the package version
- `tsargp` view options' default values
- `tsargp ...` play with option values
- `clear` clear the screen

Things to try out:

- check if similar names are suggested for unkown options (e.g., `-nof`)
- check if option names can be used as option parameters (e.g., `-ss -s`)
- check if arguments after `--` are recognized as positional, while others not
- check if a string parameter matches or fails the required regex/enums
- check if a string parameter is normalized correctly (e.g., `-ss abc`)
- check if a number parameter matches or fails the required range/enums
- check if a number parameter is normalized correctly (e.g., `-ns 1.7`)
- check if the `-b` option can be specified in combination with others
- check if value count is limited to a certain amount (e.g., `--stringsEnum one two one two`)
- check if duplicate values get removed (e.g., `--numbersEnum 1,2,1,2`)
- check if values get overridden when an option is specified repeatedly (e.g., `-s 1 -s 2`)
- check if values get appended when an option is specified repeatedly (e.g., `--numbersEnum 1 --numbersEnum 2`)
- check if a suggestion is printed when forgetting to delimit values (e.g., `--numbersEnum 1 2`)
